# EightCore
EightCore es una plataforma web integral diseñada para centralizar y digitalizar la gestión académica de la Unidad Educativa Eight Academy. Permite a docentes, estudiantes y padres de familia acceder en tiempo real al seguimiento del progreso académico, la gestión de calificaciones y el registro de novedades de conducta o desempeño, todo desde una sola interfaz segura y adaptada a cada rol.

El proyecto nace de la necesidad identificada en la institución: la ausencia de una herramienta unificada generaba desorganización en el monitoreo del progreso estudiantil, retrasos en la retroalimentación y dificultades para la comunicación entre los actores de la comunidad educativa.
